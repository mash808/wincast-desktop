## Wincast

Wincast is currently a WIP prototype for macOS. The goal of the project is to be a cross-platform, albeit much simpler, alternative to raycast.

### Features
- [x] Summon wincast from anywhere with `Cmd` + `Space`
- [ ] Launch applications
- [ ] Run system commands
